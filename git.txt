Git, GitHUB
-----------

Github
  Kliensből jelszó helyett PAT-ot kell használni
	1) Create a PAT on GitHub
		1. Go to GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic).
		2. Click Generate new token (classic).

Git témák
	- Repositories
	- staging environment (lokál branchek)
	- branch

- Kliens: git cli, git-scm.com

- clone lokál gépre
  	git clone https://github.com/barczy/Tech.git


Lokál configurálás
- mail, user név ellenőrzése (user.name user.email)
	git config --global user.name

Státusz
	git status
	git branch
	git remote -v
		Hogyan van összekapcsolva

Visszafele
- Mire rakok commitot, mit követek
	git add git.txt
	git add .
- commit
	git commit -m "first commit"
- push
	git push https://<PAT Token>@github.com/barczy/Tech.git


Branch kezelés
	git branch new-name
		új ág létrehozása
	git checkout <név>
		új ág kivétele
	git merge <branch>
		merge
	git log


Törli a lokális függőben lévő módosításokat
  git checkout -t -f origin/feature/main-docker
